# Baad me mujhe kaunsa link bhejna hai?

Agar aap real Google Sheet CRM connect karna chahte ho, to mujhe **sirf ye link** bhejna hoga:

## Apps Script Web App URL
Format aisa hota hai:

```text
https://script.google.com/macros/s/XXXX/exec
```

## Ye link kahan se milta hai?
1. Google Sheet kholo
2. Extensions → Apps Script
3. Code paste / save
4. **Deploy** → **New deployment**
5. Type: **Web app**
6. Access: **Anyone with the link**
7. Deploy
8. Jo URL milega, wahi bhejna hai

## Abhi ke liye zaroori hai?
Nahi.

Production-ready test build abhi **without this link** bhi chal jayega:
- WhatsApp flow chalega
- cart chalega
- contact form chalega
- ad editor chalega
- lead queue local mode me save hogi

Real Google Sheet save sirf tab live hoga jab aap ye URL doge.
